# Retirement Income Simulator v1.02 Interaction Polish Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ship v1.02 with concise chart tooltips, compact toggleable lump sums, independently resizable tables, stable modelling context, complete panel scrolling, and iPad-safe interactions.

**Architecture:** Preserve the dependency-free single HTML application. Advance the deterministic schema to 8, add small pure helpers for tooltip and active-withdrawal behaviour, and centralise UI snapshot/restore around existing render paths rather than adding a framework.

**Tech Stack:** HTML5, CSS Grid, CSS custom properties, vanilla JavaScript, Canvas 2D, Pointer Events, localStorage, Node.js regression harness, Playwright CLI for responsive browser verification.
